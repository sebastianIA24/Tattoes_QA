# Wakka-Tattoes 🎨



Aplicación web para la gestión y promoción de un estudio de tatuajes.  

Este proyecto busca ofrecer una plataforma sencilla donde los clientes puedan explorar diseños, reservar citas y conocer más sobre el arte del tatuaje.






## 🚀 Características principales

- Catálogo de diseños de tatuajes.

- Información sobre artistas y estilos.

- Sistema de reservas en línea.

- Sección de contacto y redes sociales.



## 🛠️ Tecnologías utilizadas

- \*\*Frontend:\*\* HTML, CSS, JavaScript (puedes especificar framework si usas React/Vue/Angular).

- \*\*Backend:\*\* Node.js / Express (ajusta según tu stack).

- \*\*Base de datos:\*\* MongoDB / MySQL (según corresponda).

- \*\*Integraciones:\*\* GitHub Actions + SonarCloud para análisis estático.
